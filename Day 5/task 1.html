PROGRAMING LANGUAGES :

A programming language is a way for programmers (developers) to communicate with computers.


The basic components of a computer are:

Input unit,Central Processing Unit(CPU),Output unit.


The CPU is further divided into three parts :

Memory unit,Control unit,Arithmetic Logic unit


Most Popular Programming Languages :

C,Python,C++,Java,C#,Swift,JavaScript


About syntax :

Syntax refers to the rules that define the structure of a language.


Imperative And Declarative :

imperative languages and declarative languages: An imperative language specifies the exact steps the computer must take to complete a desired action.
