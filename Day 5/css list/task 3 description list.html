HTML Terms

HTML :
HTML stands for Hypertext markup language.
HTML Element :
An Element in HTML usually consist of a start tag and close tag.
HTML Tag :
A piece of markup language used to indicate the beginning and end of an document.
Angle bracket :
It is Commonly used to Enclose a code of some type.
HTML Attribute :
Attributes provides additional information about HTML elements.
HTML Comment :
place a tag after the code that you want to hide.
HTML Object Element :
The object tag defines a container for an external Resource.
HTML Special Characters :
Special characters are typically those that cant't be easily typed into a Keyboard.
